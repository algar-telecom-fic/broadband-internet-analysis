Super Senha

Por Guilherme Rodrigues Tafelli da Silva, UFU Brazil

Timelimit: 1

Marquinhos SA é uma empresa de segurança muito conhecida no mundo todo. Marquinhos esta criando uma super porta que é super protegida, porém, ele não tem nenhum programador em sua empresa e para abrir a super porta é preciso de um super sistema de senha que super proteja de ladrões. Sofhia, uma amiga muito proxima de Marquinhos soube que você é programador e te indicou para ele. Obs.: Você vai receber 10 Milhões!!!

Para que o sistema seja mais seguro, Marquinhos disse que primeiramente uma senha será configurada. O usuário tem 3 changes para colocar a senha para abrir a porta, e caso exista 3 erros a porta é bloqueada e apenas a Marquinhos SA pode destravar. Já se o usuario acertar a senha a porta é destravada.

Entrada

A entrada consiste em uma string N, que representa a senha da porta.

Após configurar a senha, até 3 strings S são inseridas como tentativa do usuário.

Saída

Você deve imprimir “Porta bloqueada!”, se as 3 tentativas forem esgotadas, e “Porta liberada!”, se o usuário acertar a senha.
