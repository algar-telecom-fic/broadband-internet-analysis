Esse relatório deve receber 2 arquivos obrigatoriamente e 1 opcional, todos no formato ".csv". O primeiro arquivo contém os dados atuais das estações, os quais serão comparados com os dados do arquivo base. O segundo arquivo é justamente o arquivo das informações usadas como base, as informações anteriores das estações. Esses dois são obrigatórios. O terceiro arquivo, opcional, contém apenas duas colunas, que relacionam a Localidade com suas respectivas Regionais, e o anexo só precisa ser feito caso haja alguma localidade nova. Fora os três arquivos, a data do arquivo atual e do arquivo base devem ser especificados separadamente no campo data do formulário.
